# hthleft-lab

## 项目简介

hthleft-lab 是一个用于归档和发布独立 HTML 页面的仓库。这里存放着各类静态网页资源，涵盖工具、演示、文档或实验性页面。仓库本身不针对特定域名或网站，仅作为 HTML 文件的集中存储与分发点。

## 目录结构

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 主要页面归档目录
│   ├── example-page/   # 单个页面示例（含资源文件）
│   └── ...
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面及其附属资源（CSS、JavaScript、图片等）。
- 每个页面建议以独立子目录形式组织，目录名即页面标识。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容可包含演示、工具、文档或实验性功能，不限于特定主题。
- 页面之间相互独立，不共享状态或依赖。
- 归档页面可能不定期更新或新增，请以仓库实际内容为准。

## 维护说明

- 本仓库由维护者不定期更新，不保证所有页面长期可用或兼容。
- 若发现页面异常或链接失效，可通过 Issues 反馈。
- 暂不接受外部直接提交的页面合并请求，除非经过沟通确认。
- 仓库历史提交记录可供参考页面变更情况。

## 使用方式

1. 克隆或下载仓库到本地。
2. 直接打开 `pages/` 下对应目录中的 `index.html` 文件即可浏览页面。
3. 若通过 GitHub Pages 部署，页面可通过仓库对应的 Pages 链接访问（需自行配置）。
